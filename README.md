# DreamFrame 2.0

DreamFrameLLC is a multifaceted digital-services studio delivering AI-powered creative tools, automation, and web experiences.

## Tech Stack

- **Frontend**: Next.js 15 + TypeScript + Tailwind CSS + shadcn/ui
- **Backend**: FastAPI (Python)
- **Deployment**: Vercel (frontend) + RunPod / local GPU (API)
